Система кассового аппарата

Добро пожаловать в систему кассового аппарата для вымышленного магазина.

Доступные инструменты:

  • Панель администратора - управление товарами и просмотр чеков
  • Кассовый аппарат - API для создания чеков

Как использовать:

Для создания чека отправьте POST-запрос на "/cash_machine/" со следующей структурой:

    {
        "items": [1, 2, 3]
    }

где числа - это ID товаров.
